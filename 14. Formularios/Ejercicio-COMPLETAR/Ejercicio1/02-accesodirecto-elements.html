Selección directa de nodos del DOM

JavaScript dispone de varios métodos que nos ayudan a seleccionar elementos del DOM ya sea a través de su id, su clase, el tipo de etiqueta, la propiedad name y la más versátil de todas, un selector de CSS.

Para hacer los insertos de código en JavaScript de los ejemplos se ha utilizado la librería code-prettify

Más a cerca de code-prettify »

getElementById()

Podemos seleccionar un elemento en base a su id. Este método devuelve un solo elemento.

//seleccionamos el elemento con id="user"
const user = document.getElementById("user");
//seleccionamos un elemento con id=formulario
const formulario = document.getElementById("formulario");

getElementsByTagName()

Podemos seleccionar varios elementos en base a su etiqueta. Este método devuelve un array.

const enlaces = document.getElementsByTagName("a");
const enlace2 = enlaces[1]; //accedemos al segundo enlace

getElementsByName()

Podemos seleccionar varios elementos en base a su atributo name. Este método devuelve un array. 

const frutas = document.getElementsByName("fruta");
const fruta = fruta[1].value; 
//accedemos al segundo elemento que tiene name="fruta"

querySelector()

Podemos seleccionar un elemento con un selector CSS. Este método devuelve un solo elemento y si son varios los que concuerdan con el selector, devolverá el primero. 

//devuelve el primer elemento de la clase tab que esté dentro de un div
const tab = document.querySelector("div .tab");
//accedemos a un elemento por su id
const fechaNac = document.querySelector("#fechaNac");
//es interesante acceder por atributo
//Accedemos al primer elemento con name="meal"
const meal = document.querySelector("input[name='meal']");
//accedemos al elemento con name="meal" que está seleccionado
//para ello existe una pseudoclase :checked
const opsel = document.querySelector("input[name='meal']:checked");
console.log(opsel.value);

querySelectorAll()

Podemos seleccionar varios elementos con un selector CSS. Este método devuelve un array con todos loslementos que concuerden con el selector CSS 

//devuelve todos los elementos class="tab" que estén dentro de un div
const tabs = document.querySelectorAll("div .tab");

//es interesante acceder por atributo
//Accedemos al todos los elemento con type="checkbox"
const cks = document.querySelectorAll("input[type='checkbox']");
